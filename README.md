# FutbolRush


FutbolRush is a small portrait-oriented 2D football shooting game built with Godot 4. You control a footballer near the bottom of the pitch, dodge defenders coming from above, and score by shooting into the top-center goal.

## Run

1. Open this folder in Godot.

## Controls

- `A` or Left Arrow: move left
- `D` or Right Arrow: move right
- Enter: spacebar
- Click `Play` or `Restart` from the game menus

## Notes

Best score is saved locally with Godot's `user://` storage.
